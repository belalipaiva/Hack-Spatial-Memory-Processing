# Hack-Spatial-Memory-Processing
This repository contains a samples of electrophysiological recordings of three brain regions: 
- Basolateral amygdala;
- Dorsal Hippocampus;
- Ventral Hippocampus.

## Dataset usage restrictions

The dataset provided in this repository is **UNPLUBISHED** and is made
available **exclusively for use within the context of HACKATHON**.

The dataset:
- may be used **ONLY FOR HANDS-ON ACTIVITIES DURING THE HACKATHON (20-27)**
